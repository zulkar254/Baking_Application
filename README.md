# 🏦 Baking Application

A simple **banking simulation project** built using **Core Java**, **HTML**, **CSS**, and **JavaScript**.

---

## 🎯 Project Overview

This project demonstrates a simple banking system that allows users to:
- Deposit money 💰  
- Withdraw money 💸  
- Check balance 🧾  

It is designed as a college-level project to show how Java and basic frontend technologies can work together for a mini banking solution.

---

## 🧠 Features
✅ Deposit and Withdraw functionality  
✅ View current account balance  
✅ Simple, responsive web interface  
✅ Data stored locally using **browser LocalStorage**  
✅ Easy to use — no backend setup required  

---

## 🛠️ Tech Stack

| Layer | Technology Used |
|-------|------------------|
| Frontend | HTML, CSS, JavaScript |
| Backend Logic | Core Java (Console-based) |
| Storage | Browser LocalStorage |
| IDE | Eclipse / VS Code |
| Platform | Windows 10 / 11 |

---

## 🚀 How to Run

### 🖥️ Java Application
1. Open the project folder in **Eclipse**.  
2. Run the `Main` class from the **Baking_Application** project.  
3. Perform deposit, withdraw, and check balance operations via console.

### 🌐 Frontend Interface
1. Open the `frontend/index.html` file in your browser.  
2. Use the buttons to:
   - Deposit  
   - Withdraw  
   - Check Balance  
3. Data is saved in browser LocalStorage — no backend required!
